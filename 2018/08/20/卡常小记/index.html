方法二

有取模的可以优化，取模是 log 的。

mod 级别的数 x（加、减）： if (x > mod) x -= mod.

$mod^2$ 级别的数 x (乘、除) ：const ll mod2 = mod * mod * 1LL. if (x > mod2) x -= mod2. 循环结束时 x %= mod.
