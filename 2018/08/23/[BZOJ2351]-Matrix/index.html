imily's notes

题目链接

题意略。

矩阵哈希，对于一个位置 [x, y] 我们分别记录它的横向与纵向，记录方法与一般字符串哈希无异。

求出矩阵哈希前缀和，开一个 map 数组 mark，记录在原矩阵中哪些可能的 $a * b$ 的矩阵出现过，对于每一个询问，计算出输入矩阵的哈希值 res，如果 mark[res] > 0, 输出 1，反之输出 0.
