莫队

题目链接

这题首先可以判断是莫队啦，然后怎么做呢？

O(n) 计算块数肯定是不行的了，那么我们还是要用 O(1) 的操作代替它。

维护一个 vis[] 记录 x 是否被访问过，update 的时候看看 vis[x - 1] 和 vis[x + 1] 是否被访问过。

如果 vis[x - 1] = vis[x + 1] = 1, 那么显然块数 ans -= 1；

如果其中一个为 0，其中一个为 1，那么 ans 不变。

如果 vis[x - 1] = vis[x + 1] = 1, 那么 ans += 1.

所以就是这样。
